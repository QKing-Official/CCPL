CCPL

C Compiled Programming Language

The ease of Bash. The power of C. CCPL is a high-level scripting language that transpiles to C and compiles to a native binary — write clean, readable code and get raw performance automatically.

Get Started Browse Packages GitHub

$ ccpl hello.ccpl -o hello

Installation

One-line installer

bash <(curl -s https://raw.githubusercontent.com/
QKing-Official/CCPL/main/installer.sh)
Uses a dialog-based UI to select components: ccpl (compiler), barite (package manager), offline (local package cache). Installs to /opt/ccpl/ and symlinks into /usr/local/bin/.

Build from source

git clone https://github.com/QKing-Official/CCPL.git
cd CCPL
gcc ccpl/main.c ccpl/lexer.c ccpl/parser.c -o ccpl-bin
gcc barite/barite.c -o barite-cli
Run locally with ./ccpl-bin and ./barite-cli. To install globally, move binaries to /opt/ccpl/ and symlink to /usr/local/bin/.

Compiler flags

Flag Description
-o <name>Set output binary name (default: out)
-a, --autoAuto-install missing packages via Barite
-r, --replStart interactive REPL mode
-k, --keep-cKeep the generated .c file
--quietSuppress success output

Install packages with Barite

barite-cli install math
barite-cli install io shell
barite-cli install local math   # offline / no internet
