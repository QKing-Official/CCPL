# cnum package

C-style numeric helper methods.

## Methods

- cnum.abs(x)
- cnum.min(a, b)
- cnum.max(a, b)
- cnum.clamp(x, lo, hi)
- cnum.floor(x)
- cnum.ceil(x)
- cnum.sqrt(x)
- cnum.round(x)
- cnum.sign(x)
- cnum.is_even(x)
- cnum.gcd(a, b)
- cnum.lcm(a, b)

Runtime implementation is in src/runtime.c.
